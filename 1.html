Basic Selector

* {
    margin: 0px;
    padding: 0px;
}

Ulduz isaresi sehifedeki butun elementleri secir.Daya cox bundan sehifedeki margin ve paddingleri sifirlamaq ucun isdifade edirler

Id Selector

#conteiner {
    width: 570px;
    height: 100px;
    padding-left: 10px;
}

id atributuna css vermek ucun diyesden(#) istifade olunur

Class Selector

.conteiner {
    width: 570px;
    height: auto;
    margin: auto;
}

class atributuna css vermek ucun noqteden(.) istifade olunur

Element Selector

div {
    background-color: black;
    color: red;
}
ul {
    margin-left: 0px;
}
a {
    text-decoration: none;
    color: blue;
}

Her bir elemente bele css vermek olar

Combinator Selector

ul li {
    font-size: 10px;
    text-align:right;
}

Burda css ul taginin icindeki li tagini secir.

ul + p {
    color: red;
}

Burda css ul-den sonraki p-ni qirmizi reng edecek

div#conteiner > ul {
    width: 570px;
}

Burda css id-si conteiner olan div-in icindeki ul-leri sececek

ul ~ p {
    color: green;
}

Bu kod ul-den sonra gelen her bir p-ni qirmizi reng edecek

Atribute Selector

a[title] {
    text-decoration: none;
}

Bu kod yalniz title ozelliyine sahib olan link etiketini sececek. Title ozelliyi olmayan hec bir link etiketi bu stili almayacaq 

a[href="https://net.tutsplus.com"] {
    color: red;
}

Ancaq https://net.tutsplus.com bu adrese sahib olan linkler qirmizi reng olacaq

a[href^="http"] {
    padding-left: 10px;
}

Bu kod ancaq href-i http ile baslayan linkleri sececek

Pseudo Selectors

a:link {
    color: red;
}
a:visited {
    color: grey;
}

:link ziyaret edilmemis linkleri secir

:visited ziyaret edilmis linkleri secir

input[type="radio"]:checked {
    border: 1px solid grey;
}

Her yoxlanilmis input elementini secir

p::after {
    content: "....";
}

:: after selector seçilmiş hər elementin məzmununa bir şey əlavə edir.

a:hover {
    text-decoration: underline;
    color: blue;
}

:hover selector-u ile elementinizin ustune sicani getirdikde vermek isdediyiniz css-i kodlasdira bilersiniz 

*:not(p) {
    color: blue;
}

p tagindan basqa butun yazilar goy reng olacaq

p::firth-line {
    background-color: yellow;
}

::firth-line selector-u gosterdiyiniz secicinin birinci setrini secir

Nth Child and Type Selectors

p:nth-child(n) {
    background-color: red;
}

Bu kod p-ler arasinda n-in yerine hansi reqemi yazsaq onu secir.Meselen n-in yerine 3 yazsaq 3-cu p-nin arxa rengi qirmizi olacaq

p:nth-last-child(n) {
    background-color: red;
}

Bu selector axirdan sayaraq n-ci usagi secir.Meselen n-in yerine 2 yazsaq axirdan 2-ci usagin background-color-u qirmizi olacaq

p:nth-of-type(n) {
    background-color: grey;
}

p:nth-of-type(n) selector-u p-nin n-ci usagini secir

p:nth-last-of-type(n) {
    background-color: grey;
}

Bu selector elementin axirdan n-ci usagini secir

ul li:firth-child {
    color: greenyellow;
}

Bu selector elementin birinci usagini secir

ul li:last-child {
    color: yellow;
}

Bu selector elementin axirinci usagini secir

p:only-child {
    background-color: black;
}

Bu selector elementin yegane usagini secir

 {
    color: yellow;
}

Bu selector elementin axirinci usagini secir
